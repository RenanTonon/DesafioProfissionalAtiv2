No caso de mensageria:
Filas:São estruturas de dados onde mensagens são armazenadas temporariamente até serem consumidas, a fila segue a ordem de o primeiro a chegar é o primeiro a sair (FIFO)
Produtores: São os responsaveis por envia a informação para a fila 
Consumidores:São os responsaveis por tratarem a informação que recebeu da fila
Tópicos: são uma especie de classificador onde fica responsavel em 'sintonizar' um tipo de mensagens para um tipo de consumidor
